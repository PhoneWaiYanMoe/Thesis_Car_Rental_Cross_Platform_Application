# Mobile Application

Flutter cross-platform mobile app for iOS and Android.

## Features

- User authentication
- Browse available motorbikes
- Book and manage rentals
- Real-time notifications
- Payment integration
- Review and rating

## Tech Stack

- Flutter 
- Dart
- Firebase (FCM)
- Google Maps

## Getting Started
```bash
cd mobile
flutter pub get
flutter run
