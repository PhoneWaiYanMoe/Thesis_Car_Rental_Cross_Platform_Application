# Request Service

Request management service for the Wiz Car Rental platform.

## Features
- Create and manage support requests
- Request categorization and routing
- Approval workflows
- Status management
- Audit logging
- Event publishing via RabbitMQ

## Tech Stack
- Node.js + Express
- PostgreSQL
- RabbitMQ

## Installation
1. Install dependencies: `npm install`
2. Set up PostgreSQL database
3. Configure .env file
4. Run: `npm run dev`
