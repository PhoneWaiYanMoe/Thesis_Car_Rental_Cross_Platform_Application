# Backend Microservices

This folder contains all backend microservices for the Motorbike Rental System.

## Services

- `user-service/` - User authentication and profile management
- `vehicle-service/` - Motorbike inventory management
- `booking-service/` - Reservation and booking logic
- `payment-service/` - Payment processing
- `notification-service/` - Push notifications and emails
- `review-service/` - Ratings and reviews
- `location-service/` - GPS and map integration
- `analytics-service/` - Data analytics and reporting

## Tech Stack

- Node.js 
- PostgreSQL, MongoDB, Redis
- Docker
